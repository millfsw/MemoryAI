## Slide 2: Context

**End-User:**
Students preparing for exams who need to create study materials quickly.

**Problem:**
Manually creating flashcards and study summaries from lecture notes is time-consuming and tedious. Students spend hours reading through materials and extracting key concepts instead of actually studying them.

**Product Idea (One Sentence):**
MemoryAI uses artificial intelligence to automatically generate flashcards, summaries, and quizzes from any text or document — turning hours of manual preparation into seconds.

---

## Slide 3: Implementation

**How We Built It:**
- **Backend:** FastAPI (Python) with JWT authentication, OpenRouter AI API integration
- **Database:** PostgreSQL for storing decks, flashcards, user progress
- **Frontend:** React + TypeScript + Vite with responsive UI
- **Deployment:** Docker Compose on Ubuntu 24.04 VM, nginx reverse proxy

**Version 1 (MVP):**
- Text-based flashcard generation via AI
- Save and view generated decks
- User registration and authentication
- Study mode with flashcard flipping

**Version 2 (Improvements):**
- File upload support (PDF, DOCX, TXT)
- AI Chat Sidebar — ask questions about your material
- Quiz Mode with multiple-choice questions
- Study Mode with progress tracking (Known / Don't Know)
- Deck management (search, delete, select multiple)
- Markdown-formatted AI responses
- Full Docker deployment on VM
