📋 Context

👤 End-User

University students preparing for exams who need to create study materials quickly and efficiently.

⚠️ Problem

Manually creating flashcards from lecture notes is time-consuming. Students spend hours extracting key concepts instead of actually studying.

💡 The Solution

MemoryAI uses artificial intelligence to automatically generate flashcards, summaries, and quizzes from any text or document — turning hours of manual preparation into seconds.

🛠️ Implementation

Version 1 — MVP
  • Text-based flashcard generation via AI
  • Save and view generated decks
  • User registration & authentication
  • Study mode with flashcard flipping
  • Summary generation
Version 2 — Expanded
  • File upload (PDF, DOCX, TXT)
  • AI Chat Sidebar — contextual Q&A
  • Quiz Mode with multiple-choice
  • Study Mode with progress tracking
  • Deck management (search, delete)
  • Markdown AI responses
  • Docker deployment on VM

Tech Stack

Backend: FastAPI (Python) + JWT + OpenRouter AI  |  Database: PostgreSQL  |  Frontend: React + TypeScript + Vite  |  Deploy: Docker Compose + nginx on Ubuntu 24.04
